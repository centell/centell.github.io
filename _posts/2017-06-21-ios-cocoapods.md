iOS 개발시 쓰는 library 관리 툴인 듯.

설치 : sudo gem install cocoapods  
삭제 : sudo gem uninstall cocoapods  
초기화 : pod init  
라이브러리 프로젝트에 추가 : pod install  
라이브러리 변경점 업데이트 : pod update  
지원 라이브러리 검색 : pod search 라이브러리명
